# 🔥 CrazyWeapons - Minecraft Plugin

![Minecraft](https://img.shields.io/badge/Minecraft-1.21.8-brightgreen)
![Version](https://img.shields.io/badge/Version-1.1.0-blue)
![Java](https://img.shields.io/badge/Java-21-orange)

> **Unleash chaos with insane custom weapons!** This plugin adds 10 unique crazy weapons with special abilities that will transform your Minecraft experience.

---

## ⚔️ Weapons

### 💥 Explosive Sword
- **Material:** Diamond Sword
- **Ability:** Creates massive explosions on every hit!
- **Command:** `/cw explosive`
- **Perfect for:** Clearing mobs and making a dramatic entrance

### ⚡ Lightning Staff
- **Material:** Blaze Rod
- **Ability:** Right-click to summon lightning strikes at your target location
- **Command:** `/cw lightning`
- **Perfect for:** Long-range devastation and feeling like a god

### ❄️ Freeze Ray
- **Material:** Stick
- **Ability:** Freezes enemies in place with extreme slowness
- **Command:** `/cw freeze`
- **Perfect for:** Crowd control and stopping fast enemies

### ✦ Teleport Wand
- **Material:** Ender Pearl
- **Ability:** Right-click to instantly teleport to where you're looking (up to 30 blocks)
- **Command:** `/cw teleport`
- **Perfect for:** Quick escapes and parkour shortcuts

### 🩸 Vampire Blade
- **Material:** Netherite Sword
- **Ability:** Steals 50% of damage dealt as health
- **Command:** `/cw vampire`
- **Perfect for:** Sustain in long battles and feeling immortal

### 🌪️ Tornado Axe
- **Material:** Golden Axe
- **Ability:** Launches ALL nearby enemies (6 block radius) into the air on hit!
- **Command:** `/cw tornado`
- **Perfect for:** Crowd control and sending mobs flying

### ☠️ Poison Dagger
- **Material:** Iron Sword
- **Ability:** Applies strong Poison III on every hit
- **Command:** `/cw poison`
- **Perfect for:** Slowly melting enemies over time

### 🔥 Fire Hammer
- **Material:** Iron Axe
- **Ability:** Sets enemies on fire for 10 seconds on hit
- **Command:** `/cw fire`
- **Perfect for:** Burning everything in sight

### 🔮 Gravity Mace
- **Material:** Mace
- **Ability:** Right-click to pull all nearby enemies (10 block radius) toward you
- **Command:** `/cw gravity`
- **Perfect for:** Gathering enemies before unleashing another weapon

### 💀 Soul Reaper
- **Material:** Netherite Hoe
- **Ability:** Applies Wither II + Blindness on hit
- **Command:** `/cw reaper`
- **Perfect for:** Draining enemies from the inside out

---

## 📦 Installation

1. **Download** the latest `CrazyWeapons.jar` from releases
2. **Place** the jar file in your server's `plugins/` folder
3. **Restart** your server
4. **Enjoy** the chaos!

---

## 🛠️ Building from Source

### Prerequisites
- Java 21 or higher
- Maven

### Build Steps
```bash
git clone <your-repo-url>
cd CrazyWeapons
mvn clean package
```

The compiled jar will be in `target/CrazyWeapons.jar`

## 🎮 Commands

| Command | Aliases | Description |
|---------|---------|-------------|
| `/crazyweapons` | `/cw` | Shows list of available weapons |
| `/cw explosive` | - | Get the Explosive Sword |
| `/cw lightning` | - | Get the Lightning Staff |
| `/cw freeze` | - | Get the Freeze Ray |
| `/cw teleport` | - | Get the Teleport Wand |
| `/cw vampire` | - | Get the Vampire Blade |
| `/cw tornado` | - | Get the Tornado Axe |
| `/cw poison` | - | Get the Poison Dagger |
| `/cw fire` | - | Get the Fire Hammer |
| `/cw gravity` | - | Get the Gravity Mace |
| `/cw reaper` | - | Get the Soul Reaper |
